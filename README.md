# Fullstack-project

## Arquitetura do projeto

Na parte de back-end utilizei NodeJS, Express, Prisma e Cors.

Na parte de front-end utilizei React, Vite e Axios.

No banco de dados utilizei PostgreSQL.

## Requisitos para rodar o projeto

Baixar softwares, libraries e frameworks citados na arquitetura do projeto.

## Como rodar este projeto

Rodar o comando "npm run dev" no terminal da pasta back-end e no terminal da pasta frontend.

## Porque criei este projeto

Esse projeto foi feito para controle de produtos de um mercado.
